Mysql版本	 	===>  mysql-5.6.38-winx64

phpMyAdmin版本	===>  phpMyAdmin-4.8.0-all-languages


Php版本	 	 	===>  php-5.6.35-Win32-VC11-x64

Php-Redis扩展  	===>  php_redis-2.2.7-5.6-ts-vc11-x64

Redis版本		===>  Redis-win-x64-3.0.504
